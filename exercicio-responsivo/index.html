Sobre o curso

Neste curso de Programação Front-End, você aprenderá a usar usar a biblioteca React.js, em conjunto com as tecnologias HTML, CSS e JavaScript, para programar a interface do usuário de aplicações ricas e complexas.

Ao final desse curso, você vai:

Criar interfaces dinâmicas para sites e web apps, entender desde os fundamentos até habilidade avançadas em front-end, além de conhecer sobre boas práticas em codificação, testes de software e como buscar vagas no mercado.
